为什么是 Moonshine？

不是云端转写的廉价替代品，而是重新定义本地会议记录体验的产品。

🎙️
实时双路采集
同时捕获麦克风音频和系统内录（Windows WASAPI Loopback / macOS BlackHole），区分「你说」和「对方说」，输出结构化对话记录。
⚡
端到端 <300ms
160ms 音频 chunk + ~80ms moonshine 推理，含 VAD 跳过静音。实时字幕无感知延迟，会议节奏不被打破。
🔒
真正本地运行
所有数据留在本地磁盘，不经过任何服务器。适合处理敏感会议、知识产权讨论、完全没有网络的离线环境。
💾
多格式一键导出
一键导出 Markdown（会议纪要）、SRT 字幕（视频压制）、纯文本（快速复制），兼容所有主流笔记和协作工具。
📊
VAD 静音检测
智能人声检测，自动跳过静音片段，减少 60% 无效转写。仅在有人说话时消耗算力，CPU 占用更低。
🪟
跨平台桌面应用
基于 Electron，一套代码支持 Windows / macOS / Linux。安装包一键安装，无需手动配置 Python 环境。
