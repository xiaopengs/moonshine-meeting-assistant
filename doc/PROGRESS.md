### chunk 大小重算（已执行）

```
目标：端到端 < 300ms
moonshine 推理：~80ms
buffer 余量：~20ms
→ chunk = 200ms → 取 160ms（安全余量）
→ 160ms × 16000 Hz = 2560 samples
已写入：transcribe.py  CHUNK_DURATION=0.16
```
