## 由此引发的技术调整

### chunk 大小重算

```
目标：端到端 < 300ms
moonshine 推理：~80ms
buffer 余量：~20ms
→ chunk 最大时长 = 300 - 80 - 20 = 200ms → 取 160ms
→ 160ms × 16000 Hz = 2560 samples / chunk
```
